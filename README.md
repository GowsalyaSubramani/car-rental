
## Installation and Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/car-rental-app.git
   cd car-rental-app
2. **Install dependencies:**
   ```bash
   npm install
3. **Start the development server:**
  ```bash
  npm run dev

4. **Open the application in your browser:**
    http://localhost:5173
